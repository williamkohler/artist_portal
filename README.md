# Artist Portal

## Getting started

To get started with the app, clone the repo and then install the needed gems:

```
$ bundle install
```

Migrate the database:

```
$ rails db:migrate
```

Seed the database:

```
$ rails db:seed
```

Run the test suite to verify that everything is working correctly:

```
$ rails test
```

Finally run the app on a local server:

```
$ rails server
```
